# 搜索增强功能设计

**日期：** 2026-05-31
**状态：** 已批准

## 问题背景

当前资源搜索功能存在以下问题：
1. 搜索结果中的链接可能已失效，用户无法提前知道
2. 用户无法预览分享链接中的文件内容
3. 任务管理中无法快速搜索并替换失效的链接

## 设计目标

1. 搜索结果展示时自动校验链接有效性
2. 点击搜索结果可查看分享链接中的文件列表
3. 任务编辑中支持搜索并替换分享链接

## 详细设计

### 功能 1：搜索结果链接有效性校验

**触发时机：** 搜索结果展示时自动批量校验

**前端改动：**
- `web/src/views/Search.vue`：搜索结果展示后，自动调用校验 API
- 每个结果项显示校验状态：✅ 有效 / ❌ 无效 / ⏳ 校验中
- 无效链接显示错误原因

**后端改动：**
- 已有 `GET /api/search/validate?url=...` API
- 考虑添加批量校验接口 `POST /api/search/validate_batch` 以减少请求数

**数据流：**
```
用户搜索 → 获取结果 → 并发校验所有链接 → 更新 UI 状态
```

**校验状态字段：**
```typescript
interface SearchItem {
  // ... 现有字段
  valid?: boolean | null  // null=未校验, true=有效, false=无效
  validMessage?: string   // 校验失败原因
}
```

---

### 功能 2：点击展示分享链接内容弹窗

**触发时机：** 点击搜索结果项

**前端改动：**
- `web/src/views/Search.vue`：添加点击事件，打开弹窗
- 新增 `ShareContentDialog` 组件
- 弹窗中调用 `/api/tasks/parse_share` 获取文件列表
- 弹窗底部添加操作按钮

**后端改动：**
- 已有 `POST /api/tasks/parse_share` API，无需修改

**弹窗设计：**
```
┌─────────────────────────────────────┐
│ 📁 分享内容：电影名称                │
├─────────────────────────────────────┤
│ 📄 文件1.mp4          1.2 GB       │
│ 📄 文件2.mkv          800 MB       │
│ 📁 子目录/                         │
├─────────────────────────────────────┤
│ [创建任务]  [替换任务链接]  [关闭]   │
└─────────────────────────────────────┘
```

**按钮行为：**
- **创建任务**：跳转到任务创建页面，预填分享链接和提取码
- **替换任务链接**：仅在从任务编辑进入时显示，替换当前任务的链接
- **关闭**：关闭弹窗

---

### 功能 3：任务管理中搜索资源快速更换链接

**触发时机：** 任务编辑抽屉中点击"搜索替换"按钮

**前端改动：**
- `web/src/views/Tasks.vue`：任务编辑抽屉中，分享链接输入框旁添加"搜索替换"按钮
- 点击按钮打开搜索弹窗，自动填入任务名称作为搜索关键词
- 搜索结果中点击"替换"按钮，自动更新分享链接和提取码

**后端改动：**
- 无需新增 API，复用现有接口

**交互流程：**
```
编辑任务 → 点击"搜索替换" → 打开搜索弹窗 → 选择结果 → 自动填充链接和提取码 → 保存任务
```

**搜索弹窗设计：**
```
┌─────────────────────────────────────┐
│ 🔍 搜索替换资源                      │
├─────────────────────────────────────┤
│ 搜索关键词：[电影名称]  [搜索]       │
├─────────────────────────────────────┤
│ 搜索结果：                          │
│ [✅] 电影名称 - 夸克网盘  [替换]    │
│ [❌] 电影名称 - 移动云盘  [替换]    │
└─────────────────────────────────────┘
```
